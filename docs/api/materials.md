# Materials Organization

Materials are organized by their primary category and subcategory:

- `resources/materials/raw/` - Raw materials (geological, biological, atmospheric)
- `resources/materials/processed/` - Processed basic materials (metals, alloys, polymers)
- `resources/materials/building/` - Construction materials
  - `structural/` - Load-bearing components (frames, beams, supports)
  - `functional/` - Special-purpose materials (insulation, shielding, windows)
- `resources/materials/chemicals/` - Chemical compounds and substances
