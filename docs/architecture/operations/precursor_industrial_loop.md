# Operations: Precursor Industrial Loop (Day 1)

## 1. Phase 1: The Work Camp (Provisional)
- **Primary Unit**: `car_300_lunar_deployment_robot_mk1`.
- **Infrastructure**: Inflatable habitats + 3D-printed regolith shells (for radiation/thermal protection only).
- **Economic Source**: Earth-launch manifest funded by **USD**.
- **Goal**: Establish the ISRU pipeline (Regolith -> Metals/Gases).

## 2. The Manufacturing Bridge
- **Input**: Raw Regolith + Power (40kW per CAR-300).
- **Process**: `regolith_3d_printing` technology.
- **Output**: `3d_printed_ibeam` (75kg regolith / 15kg Al-alloy).

## 3. Phase 2: The Settlement (Permanent)
- **Transition**: Occurs when the Work Camp produces enough components to initiate the `Worldhouse` Bracing phase.
- **Success State**: Transition from an Earth-dependent "Work Camp" to a self-sufficient "Settlement" trading in **GCC**.
