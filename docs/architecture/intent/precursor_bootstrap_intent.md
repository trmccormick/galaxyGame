# Intent: Precursor Mission Bootstrapping

## Overview
To solve the "Weight Penalty" of deep-space logistics, settlements must be "bootstrapped" by robotic precursor missions. This intent defines the transition from an automated landing site to a self-sustaining geological enclosure.

## Structural Philosophy
- **Units vs. Structures**: 
    - **Units**: Deployable, mobile agents (e.g., CAR-300) or portable modules (e.g., Inflatables).
    - **Structures**: Permanent geological transformations (e.g., Worldhouse).
- **The Robot-First Mandate**: Construction of "Proper Structures" is a robotic precursor task. Human arrival is gated by the establishment of a pressurized "Work Camp" environment.
- **The Biome Goal**: A Worldhouse is a **Pre-Terraforming Biome**. It moves from "Life Support" (survival) to "Ecosystem" (life) once the geological seal is verified at 100%.

## Generic Portability
All hardware and structural components must remain planet-agnostic in their core identity. Regionality (Lunar/Mars) is an environmental attribute, not a functional type.
