# Phase 3 Sprite Sheet Generation Prompt

## Sprite Atlas Specifications
- **Atlas Size:** 640 x 64 pixels
- **Sprite Count:** 10 sprites
- **Sprite Size:** 64 x 64 pixels each
- **Style:** SimCity-style isometric pixel art
- **Theme:** Mars settlement structures

## Required Sprites
1. Dome (habitat)
2. Solar panel array
3. Factory (industrial)
4. Hydroponics farm
5. Research lab
6. Power station
7. Water extractor
8. Landing pad
9. Storage module
10. Construction crane

## Art Direction
- All sprites must use isometric perspective, matching classic SimCity pixel art style
- Color palette: Mars surface tones, metallics, glass, solar blue
- Each sprite should be visually distinct and easily recognizable at 64x64 resolution
- Sprites must be arranged in a single row (atlas: 640x64)

## Usage Context
- For Phase 3 Settlement View (65536x32768 canvas, 10m/pixel scale)
- Sprites will represent buildable structures in the worldhouse placement grid and construction preview features
